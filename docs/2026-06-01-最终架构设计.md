## 二、划分原则

```
shared/  = 所有域都用的基础设施（类型、配置、指标、DB、采集器、适配器、工具函数）
monitor/ = 盯盘独有的实时逻辑（市场模式、情景引擎、风控、信号、执行、持仓、策略）
review/  = 复盘独有的盘后逻辑（管线、筛选、报告、自我进化）
stock/   = 个股分析（待开发）
```

**加了 storage/** —— 数据库和日志，gitignore。

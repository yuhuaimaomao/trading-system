## 二、可增强现有模块的思路

### 1. 三周期选股 → 增强 `strategy/screening/`

文档核心：大周期看方向 → 中周期看结构 → 小周期抓时机

- **大周期过滤器**：趋势筛选中加入周线 MA60 位置判断（「周线在60周线之上」），作为日线因子的上一层过滤
- **小周期入场确认**：盘中 scout 检测加入 60 分钟级别 MACD 金叉/底背离确认

### 2. 顺大势逆小势 → 增强 `scenario/`

文档：在大级别上升趋势里，找小级别下跌回调段入场

- 新增情景 `pullback_in_uptrend`：周线多头 + 日线缩量回踩 + 60分钟企稳
- 和现有 `add_opportunity` 的区别：后者只看日线价格区间，未结合周线趋势和60分钟确认

### 3. 时间止损 → 增强 `risk/`

文档：「设置时间止损，一个月内不涨即出」

- 持仓超过 N 个交易日未达预期涨幅 → 触发时间止损
- 实现简单，规则明确，和现有 `position_risk.py` 集成

### 4. 复盘框架对齐 → 检查 `review/` 覆盖度

文档五层复盘（宏观→板块→个股→技术→计划），逐一对照检查当前 review 输出是否覆盖。

## 三、可新增的模块/功能

### 5. 市场情绪周期检测（高优先级）

文档六阶段模型：
```
冰点 → 启动 → 加速 → 高潮 → 分化 → 冰点
```

- **新建**：`trade/scenario/market_sentiment.py` 或放在 `strategy/` 下
- **输入**：涨停家数、炸板率、连板高度、成交量变化等
- **输出**：当前情绪阶段标签 → 影响全局仓位建议
- **AI 参考**：口诀「冰点不动，启动试水，加速重仓，高潮减仓，退潮空仓」可直接写入 prompt

### 6. 主力行为学形态检测

文档：
> 牛市缓涨急跌是洗盘，熊市缓跌急涨是诱多

- **新建**：`trade/detect/` 下新增价格行为检测器
- 两个信号：`slow_rise_sudden_drop`（洗盘→加仓机会）、`slow_drop_sudden_rise`（诱多→减仓）
- 判断逻辑：对比近期涨跌速率和单日波动幅度

### 7. 半仓滚动交易 → 仓位管理模块

文档：初始建仓半仓 → 下跌加仓 → 上涨减仓 → 维持 30%-70%

- **位置**：`trade/exec/` 或 `risk/` 下
- **依赖**：需要和市场情绪周期判断联动（震荡市用滚动，趋势市用持有）
- 涉及动态仓位调度逻辑

### 8. 3浪风险识别

文档：3浪往往放量急跌，不宜盲目抄底，应防守等待

- 检测到可能的第3浪下跌时，禁用 `add_opportunity` 买入建议
- 和现有「深跌等待反弹」规则配合：如果是3浪，建议直接减仓而非等待

### 9. 情绪票 vs 趋势票分类 → AI prompt

文档定义了两类票的核心特征，可写入 AI 选股 system prompt，帮助 AI 区分机会类型。

## 四、优先级排序

| 优先级 | 方向 | 理由 |
|--------|------|------|
| 1 | 时间止损 | 最简单，规则明确，独立，收益高 |
| 2 | 市场情绪周期 | 填补系统空白，全局仓位指导 |
| 3 | 三周期共振入场 | 和现有体系最契合，增强胜率 |
| 4 | 3浪风险识别 | 防止深跌陷阱，增强风控 |
| 5 | 主力行为学形态检测 | 有价值的增量信号 |
| 6 | 半仓滚动交易 | 依赖情绪周期判断，需前置 |
| 7 | 情绪/趋势票分类 | prompt 层面改动，成本低 |
| 8 | 复盘框架对齐检查 | 查漏补缺 |

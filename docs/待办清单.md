# 待办清单

## 个股行情采集器替换 (东财 → 腾讯+同花顺)

**背景**: 东方财富 API 反爬日趋严格，`push2.eastmoney.com` 直连频繁被封。

**方案**: 三层融合采集器，文件 `data/collect/market/stock_basic_fusion.py`，数据存临时表 `stock_basic_fusion`。

**数据源**:
- 腾讯 `qt.gtimg.cn` — 基础行情 + 量比 + 振幅 + PE/PB/市值/换手率 (全市场, ~2s)
- 同花顺 `stock_fund_flow_individual` — 主力资金 (全市场, ~13s)
- 降级: 腾讯 `getBoardRankList` — 主力资金 (仅 4597 只, 缺科创/北交)

**运行方式**:
```bash
python -c "from data.collect.market.stock_basic_fusion import StockBasicFusionCollector; StockBasicFusionCollector().fetch_and_save()"
```

**2026-06-08 初版验证结果**:
- 总量: 5854 vs 旧 5858 (差4只, 可忽略)
- 量比: 5511/5513 完全一致 ✅
- PE: 3946/3977 完全一致 ✅
- 主力资金: 同花顺 vs 东财算法不同, 系统性差异 ⚠️
